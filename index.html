1. Contexto Real y Fundamento de los Algoritmos

Este sistema interactivo demuestra la aplicación directa de dos pilares de la matemática discreta en entornos computacionales y prácticos del mundo moderno:

Sucesión de Fibonacci

Fórmula de crecimiento exponencial controlado. En el plano real modela la distribución geométrica biológica, espirales orgánicas de plantas y optimización de capital mediante planes de ahorro escalonados, donde la acumulación de fondos sigue la inercia de la suma de los dos períodos inmediatos anteriores.

Números Primos

Bloques de construcción numéricos indivisibles. Fundamento indispensable de la seguridad criptográfica moderna y validación lúdica de accesos. Un entero es primo si carece de factores de descomposición lineales exceptuando el valor unitario y su propia magnitud.

Fibonacci Finanzas

Plan de Ahorro Progresivo

Simula una estrategia financiera de depósitos variables controlados por la secuencia matemática pura, calculando de forma acumulativa el patrimonio neto.

algoritmo_ahorro.js
let a = 0, b = 1;
for (let i = 1; i <= meses; i++) {
    let cuotaActual = b;
    acumuladorTotal += cuotaActual;
    let c = a + b;
    a = b; b = c;
}

Validación de Claves de Seguridad

Verifica el nivel de cifrado básico de un código de acceso analizando su primalidad mediante el recuento aritmético exacto de factores de división.

analisis_primalidad.js
let divisores = 0;
for (let i = 1; i <= codigo; i++) {
    if (codigo % i === 0) divisores++;
}
let esPrimo = (divisores === 2);
